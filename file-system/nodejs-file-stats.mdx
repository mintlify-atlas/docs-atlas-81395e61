---
title: 'Node.js file stats'
description: 'Learn how to inspect file metadata in Node.js using the fs.stat() method, including checking file type, size, and using both callback and promise-based APIs.'
---

Every file comes with a set of details that we can inspect using Node.js. In particular, using the `stat()` method provided by the [`fs` module](https://nodejs.org/api/fs.html).

You call it passing a file path, and once Node.js gets the file details it will call the callback function you pass, with 2 parameters: an error message, and the file stats:

<Tabs>
  <Tab title="ESM">
    ```javascript
    import fs from 'node:fs';

    fs.stat('/Users/joe/test.txt', (err, stats) => {
      if (err) {
        console.error(err);
      }
      // we have access to the file stats in `stats`
    });
    ```
  </Tab>
  <Tab title="CJS">
    ```javascript
    const fs = require('node:fs');

    fs.stat('/Users/joe/test.txt', (err, stats) => {
      if (err) {
        console.error(err);
      }
      // we have access to the file stats in `stats`
    });
    ```
  </Tab>
</Tabs>

Node.js also provides a sync method, which blocks the thread until the file stats are ready:

<Tabs>
  <Tab title="ESM">
    ```javascript
    import fs from 'node:fs';

    try {
      const stats = fs.statSync('/Users/joe/test.txt');
    } catch (err) {
      console.error(err);
    }
    ```
  </Tab>
  <Tab title="CJS">
    ```javascript
    const fs = require('node:fs');

    try {
      const stats = fs.statSync('/Users/joe/test.txt');
    } catch (err) {
      console.error(err);
    }
    ```
  </Tab>
</Tabs>

## What information is available

The file information is included in the `stats` variable. A lot is available, including:

- Whether the file is a directory or a file, using `stats.isFile()` and `stats.isDirectory()`
- Whether the file is a symbolic link using `stats.isSymbolicLink()`
- The file size in bytes using `stats.size`

There are other advanced methods, but the bulk of what you'll use in your day-to-day programming is this.

<Tabs>
  <Tab title="ESM">
    ```javascript
    import fs from 'node:fs';

    fs.stat('/Users/joe/test.txt', (err, stats) => {
      if (err) {
        console.error(err);
        return;
      }

      stats.isFile(); // true
      stats.isDirectory(); // false
      stats.isSymbolicLink(); // false
      console.log(stats.size); // 1024000 //= 1MB
    });
    ```
  </Tab>
  <Tab title="CJS">
    ```javascript
    const fs = require('node:fs');

    fs.stat('/Users/joe/test.txt', (err, stats) => {
      if (err) {
        console.error(err);
        return;
      }

      stats.isFile(); // true
      stats.isDirectory(); // false
      stats.isSymbolicLink(); // false
      console.log(stats.size); // 1024000 //= 1MB
    });
    ```
  </Tab>
</Tabs>

## Using the promise-based API

You can also use the promise-based `fsPromises.stat()` method offered by the `fs/promises` module:

<Tabs>
  <Tab title="ESM">
    ```javascript
    import fs from 'node:fs/promises';

    try {
      const stats = await fs.stat('/Users/joe/test.txt');
      stats.isFile(); // true
      stats.isDirectory(); // false
      stats.isSymbolicLink(); // false
      console.log(stats.size); // 1024000 //= 1MB
    } catch (err) {
      console.log(err);
    }
    ```
  </Tab>
  <Tab title="CJS">
    ```javascript
    const fs = require('node:fs/promises');

    async function example() {
      try {
        const stats = await fs.stat('/Users/joe/test.txt');
        stats.isFile(); // true
        stats.isDirectory(); // false
        stats.isSymbolicLink(); // false
        console.log(stats.size); // 1024000 //= 1MB
      } catch (err) {
        console.log(err);
      }
    }
    example();
    ```
  </Tab>
</Tabs>
