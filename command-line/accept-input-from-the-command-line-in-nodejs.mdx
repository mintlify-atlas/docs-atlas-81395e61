---
title: 'Accept input from the command line in Node.js'
description: 'Learn how to make a Node.js CLI program interactive using the built-in readline module to accept user input from the terminal.'
---

How do you make a Node.js CLI program interactive?

Node.js since version 7 provides the [`readline` module](https://nodejs.org/docs/latest-v22.x/api/readline.html) to do exactly this: get input from a readable stream such as the `process.stdin` stream, which during the execution of a Node.js program is the terminal input, one line at a time.

<Tabs>
  <Tab title="ESM">
    ```javascript
    import readline from 'node:readline';

    const rl = readline.createInterface({
      input: process.stdin,
      output: process.stdout,
    });

    rl.question(`What's your name?`, name => {
      console.log(`Hi ${name}!`);
      rl.close();
    });
    ```
  </Tab>
  <Tab title="CJS">
    ```javascript
    const readline = require('node:readline');

    const rl = readline.createInterface({
      input: process.stdin,
      output: process.stdout,
    });

    rl.question(`What's your name?`, name => {
      console.log(`Hi ${name}!`);
      rl.close();
    });
    ```
  </Tab>
</Tabs>

This code asks the user's name, and once the text is entered and the user presses enter, it sends a greeting.

The `question()` method shows the first parameter (a question) and waits for user input. It calls the callback function once enter is pressed. In this callback function, the readline interface is closed.

<Tip>
`readline` offers several other methods. Check them out in the [package documentation](https://nodejs.org/docs/latest-v22.x/api/readline.html).
</Tip>

<Note>
If you need to prompt for a password, it's best not to echo it back, but instead show a `*` symbol.
</Note>
